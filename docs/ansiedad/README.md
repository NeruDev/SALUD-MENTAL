# Ansiedad

Este módulo explora qué es la ansiedad, sus causas, manifestaciones y cómo manejarla de forma saludable.

## Contenido

- [Teoría](teoria.md): fundamentos y conceptos clave sobre la ansiedad.
- [Ejercicios](ejercicios.md): técnicas y ejercicios prácticos para reducir la ansiedad.

## Introducción

La ansiedad es una respuesta natural del cuerpo ante situaciones percibidas como amenazantes o inciertas.
En niveles moderados, puede ser útil (nos mantiene alerta), pero cuando se vuelve persistente o
desproporcionada puede afectar significativamente nuestra calidad de vida.

Comprender la ansiedad es el primer paso para aprender a gestionarla de manera efectiva.
