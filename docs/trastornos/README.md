# Trastornos

Este módulo presenta información educativa sobre trastornos de salud mental frecuentes,
con enfoque de comprensión, detección temprana y búsqueda de apoyo oportuno.

## Contenido

- [Ansiedad](ansiedad.md): tipos, síntomas, factores de riesgo y abordajes recomendados.
- [Depresión](depresion.md): características, señales de alerta, tratamientos y mitos frecuentes.
- [Estrés postraumático](estres_postraumatico.md): síntomas nucleares y estrategias de atención.
- [Adicciones](adicciones.md): factores de riesgo, señales y rutas de apoyo.

## Enfoque de este módulo

- Lenguaje claro y no estigmatizante.
- Información orientada a la prevención y al acompañamiento responsable.
- Recomendación explícita de consulta profesional ante síntomas persistentes o crisis.

## Nota importante

Este contenido no reemplaza diagnóstico ni tratamiento clínico.
Si existe riesgo inmediato para tu seguridad o la de otra persona,
contacta emergencias o líneas de apoyo en tu país.
