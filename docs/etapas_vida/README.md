# Etapas de vida

Este módulo aborda cómo cambian las necesidades de salud mental a lo largo del ciclo vital.

## Contenido

- [Infancia](infancia.md)
- [Adolescencia](adolescencia.md)
- [Adultez](adultez.md)
- [Adulto mayor](adulto_mayor.md)

## Enfoque

Cada etapa presenta desafíos y recursos distintos.
El objetivo es ofrecer orientación educativa para prevención,
detección temprana y acompañamiento respetuoso.
