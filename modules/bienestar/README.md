# Bienestar

Este módulo reúne prácticas de promoción de salud mental para la vida cotidiana.

## Contenido

- [Autocuidado](autocuidado.md): principios, evaluación personal y plan semanal.
- [Hábitos saludables](habitos_saludables.md): sueño, movimiento, nutrición, vínculos y uso digital.
- [Regulación emocional](regulacion_emocional.md): técnicas prácticas para manejar estrés y emociones intensas.

## Enfoque

El bienestar mental no significa estar bien todo el tiempo. Significa desarrollar recursos
para sostenerse en periodos difíciles, pedir ayuda cuando hace falta y cuidar hábitos que
protegen la salud psicológica a mediano y largo plazo.

## Nota importante

Este contenido es educativo. Si atraviesas síntomas persistentes o una crisis, busca apoyo profesional.
